## Инициализация

Для работы должeн быть установлен [node.js](https://nodejs.org/en/)

В папке с исходниками надо установить зависимости, выполнив:
``` bash
npm install
```


## Тестирование

Запуск локального сервера с live reload:
``` bash
npm run serve
```

## Сборка

Сборка в папку dist/:
``` bash
npm run build
```
